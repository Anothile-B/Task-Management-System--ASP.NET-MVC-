# Task Management System (ASP.NET MVC)

# Overview

The Task Management System is a professional web application built using C# and ASP.NET MVC. 
It allows teams or individuals to manage tasks efficiently by:
- Creating, editing, and deleting tasks
- Assigning tasks to employees
- Tracking task status and progress
- Highlighting overdue tasks
- Adding comments for task discussions

The app features a modern, responsive UI with a dashboard view showing task statistics,
making it suitable for demonstrating your software development skills and UI design capabilities in a portfolio.

---

# Features

- Add, Edit, Delete Tasks: Full CRUD functionality for tasks.  
- Assign Tasks: Assign tasks to employees from a dropdown.  
- Task Priority & Status: Highlight tasks based on urgency and completion.  
- Progress Tracking: Visual progress bar for each task.  
- Comments:  Add comments for task discussion.  
- Responsive UI: Modern Bootstrap 5 cards and stats dashboard.  
- Stats Overview: Dashboard shows total tasks, completed, pending, and overdue counts.  


# Technologies

- C#: Backend logic and task management.  
- ASP.NET MVC: Application structure and routing.  
- Bootstrap 5: Responsive, modern UI.  
- In-Memory Storage: No database required; tasks are stored in application memory.
